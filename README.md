# Faroe Islands Public Transport Planner

An unofficial journey planner for the Faroe Islands public transport network, covering buses, ferries, and airport services operated by [Strandfaraskip Landsins (SSL)](https://ssl.fo).

> **⚠ Not an official website.** This is an independent tool built for convenience. Timetables may be incomplete or out of date. Always verify your journey on the [official SSL website](https://ssl.fo) before travelling.

## What it does

- Plan journeys between stops across the Faroe Islands
- View the full route network on an interactive map
- Browse all bus and ferry routes with links to official timetables

## Data

Schedule data is sourced from SSL's published timetables and converted to [GTFS format](https://gtfs.org). It is not automatically updated — there may be gaps or errors.
